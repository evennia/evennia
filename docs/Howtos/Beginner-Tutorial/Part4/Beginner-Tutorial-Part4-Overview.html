Part 4: Using What We Created¶

Beginner Tutorial Parts

  • Introduction
    Getting set up.

  • Part 1: What We Have
    A tour of Evennia and how to use the tools, including an introduction to Python.

  • Part 2: What We Want
    Planning our tutorial game and what to consider when planning your own.

  • Part 3: How We Get There
    Getting down to the meat of extending Evennia to make your game.

  • Part 4: Using What We Created
    Building a tech-demo and world content to go with our code.

  • Part 5: Showing the World
    Taking our new game online and letting players try it out.

We now have the code underpinnings of everything we need. We have also tested the various components and has a simple tech-demo to show it all works together. But there is no real coherence to it at this point - we need to actually make a world. In part four we will expand our tech demo into a more full-fledged (if small) game by use of batchcommand and batchcode processors.
