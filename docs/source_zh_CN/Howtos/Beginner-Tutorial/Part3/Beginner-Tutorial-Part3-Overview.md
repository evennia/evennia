在 Evennia 初学者教程的第三部分中，我们将开始实际创建我们的教程游戏 _EvAdventure_，基于 [Knave](https://www.drivethrurpg.com/product/250888/Knave) RPG 规则集。

如果您已按照本教程系列的前几部分，您将对 Python 有一些了解，并知道在哪里找到和使用 Evennia 中的东西。我们也对将要创建的游戏类型有一个良好的概念。

即使这不是您感兴趣的游戏风格，跟随本教程也将使您获得大量使用 Evennia 的经验，并对将来自行开发很有帮助！ EvAdventure 游戏代码也经过设计，可以轻松扩展。

在这一部分制作的所有完整代码示例可以在 [evennia/contrib/tutorials/evadventure](../../../api/evennia.contrib.tutorials.evadventure.md) 包中找到。以下有三种常见的学习方式：

1. 按顺序跟随教程课程，利用它编写自己的代码，并将现成的代码作为额外帮助、上下文或检查自己的“答案”。
2. 浏览该包中的代码，并参考每部分的教程课程以获取您所看到的更多信息。
3. 以上两者的结合。

您选择哪种方式因人而异——我们每个人的学习方式都不同。

无论哪种方式，这都是一个重要的部分。您将看到大量代码，还有许多课程需要学习。毕竟我们要从头开始制作一个完整的游戏。慢慢来！
