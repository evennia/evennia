# 脚本示例

由 Griatch 贡献，2012年

这是一个用于测试的示例脚本。该脚本添加了一个简单的计时器，让你的角色在不规则的间隔内进行小的口头观察。

要测试，请在游戏中使用：

    > script me = contrib.tutorials.bodyfunctions.BodyFunctions

## 注意事项

使用 `scripts me` 可以查看正在运行的脚本。请注意，尽管计时器会倒计时至 0，但你**不会**在每个滴答声中看到回声（是否给出回声是随机的）。
