# 教程镜子

由 Griatch 贡献于 2017 年

这是一个简单的镜子对象，用于实验。它会对被查看时做出响应。

- 回显查看它的对象的描述
- 回显发送到其 `.msg` 的任何内容——如果提供了发送者，则回显给发送者，否则回显给镜子的所在位置。

## 安装

通过以下命令创建镜子：

```
create/drop mirror:contrib.tutorials.mirror.TutorialMirror
```

然后查看它。
